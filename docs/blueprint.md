# **App Name**: Soulmatter

## Core Features:

- Vector-Powered Semantic Matchmaking: Utilizes AI embeddings to calculate compatibility between users based on text-based interests and personality traits.
- Soul Vector Generation Tool: An AI reasoning tool that processes multi-step onboarding data to create a high-dimensional compatibility score and explain semantic overlaps.
- Interactive Personality Sliders: Engaging UI sliders to map personality traits such as Introvert ↔ Extrovert with real-time visual feedback.
- Spotify Metadata Profiling: A premium integration UI for connecting Spotify data to the semantic profile, visualized via custom artist cards.
- Dynamic Compatibility Visualization: Displays match scores using colorful gradient progress rings and shared interest tag cloud visualizations.
- Intelligence Dashboard: A comprehensive match dashboard featuring glassmorphism cards for match overviews, saved profiles, and profile analytics.
- Framer Motion Onboarding Flow: A polished, high-fidelity multi-step onboarding process with smooth page transitions and floating vector background effects.

## Style Guidelines:

- Primary color: Electric Violet (#9F75FF), chosen to represent futuristic technology and deep personal intuition.
- Background color: Deep Space Black (#0E0C12), a dark-mode base with subtle purple undertones to enhance contrast with glowing accents.
- Accent color: Neon Azure (#547AFF), used for success states and secondary calls-to-action, creating a cool analogous contrast with the primary violet.
- Headlines: 'Space Grotesk' for a computerized, scientific, and modern feel. Body text: 'Inter' for a clean, neutral, and machined look that provides high readability across devices.
- Utilize Lucide icons with thin weights to maintain a premium 'Linear' and 'Notion' inspired aesthetic, emphasizing minimalist utility.
- Glassmorphic design using backdrop-blur-md and subtle white-tinted borders for container cards, layered over a network-visualization background.
- Smooth, extensive motion transitions using Framer Motion including card hovers, spring-based scroll reveals, and liquid background gradients.
